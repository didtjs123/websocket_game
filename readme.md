이 프로젝트는 WebSocket을 활용하여 **클라이언트와 서버 간 실시간 데이터 통신**을 구현했습니다.

---

## **📝 주요 기능**

### 🎮 **1. 스테이지 구분**
- 스테이지는 아래의 기준으로 구분됩니다:
  - **점수**: 특정 점수를 달성하면 다음 스테이지로 이동.

### 🏆 **2. 스테이지별 점수 구분**
- 각 스테이지에서 획득하는 점수는 개별 규칙에 따라 계산됩니다.
- 예시:
  - **1스테이지**: 1초당 **1점**.
  - **2스테이지**: 1초당 **2점**.
    ...
    ...
